# C++编译器前端生成器（lex yacc）

#### 介绍
C++编译器前端生成器，只有两个文件，automata.cpp,automata.h。按照编译器领域经典教科书龙书上描述的算法编写的编译器前端生成器，包含词法分析器生成器和文法分析器生成器。支持LR(0)和LR(1)两种文法分析方式。
